# GDOT ATSPM Analysis Automation

Python-based OCR and computer vision workflows for extracting GDOT ATSPM traffic signal performance metrics from chart-based visualizations.

## What this project does
- Automates extraction of Split Monitor and Left Turn Gap metrics
- Converts chart-based traffic signal performance reports into structured datasets
- Supports downstream signal timing, split utilization, and gap availability analysis

## Main components
- `split_monitor/` batch download and extraction workflows
- `left_turn_gap/` batch download and extraction workflows
- `docs/` protocols and processing notes

## Skills Demonstrated
- Python automation
- OCR-based data extraction
- Computer vision for chart interpretation
- Data pipeline design
- Traffic operations analytics support
